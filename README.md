# 战锤史记

> 以《史记》纪传体例重写战锤（40K + 中古战锤）历史
>
> 106篇 · 约38万字 · 本纪、世家、列传、表、书五体俱全

---

## 这是什么

战锤（Warhammer）的官方背景故事分散在三十多年的小说、规则书和游戏设定中，信息量巨大且碎片化。《战锤史记》用中国《史记》的纪传体例重新编排战锤故事——把关键人物和势力的故事用清晰的中文单独成篇，像读历史人物传记一样逐步建立对战锤世界的认知。

## 包含什么

两个独立的战锤宇宙：

| 宇宙 | 背景 | 核心内容 |
|------|------|----------|
| **战锤40K** | 第41个千年的黑暗科幻 | 人类帝国、混沌诸神、二十位基因原体、银河各方异形势力 |
| **中古战锤** | 旧世界的剑与魔法 | 人类帝国、精灵、矮人、亡灵、混沌，直至终焉之时 |

两个世界共享混沌四神（恐虐、奸奇、纳垢、色孽）的设定，但故事和因果各自独立、互不交叉。

## 体例说明

采用《史记》五体结构：

- **本纪**：帝皇、基因原体、西格玛、纳加什等核心人物
- **世家**：人类帝国、混沌、灵族、矮人等势力的兴衰
- **列传**：马卡多、阿巴顿、亚瑞克、哥特列克等英雄或反面人物
- **表**：世系表、年表、功勋录——按时间或谱系维度组织信息
- **书**：亚空间、灵能、魔法八风、机械教科技等专题解读

每篇文章包含：概述 → 出身/起源 → 事迹 → 结局/现状 → 简要评述。

## 阅读建议

1. 先读 `00-综述.md`，建立对两个战锤世界的整体认知
2. 挑感兴趣的人或势力开始读——每篇独立可读，不需要按顺序
3. 专有名词首次出现附英文原名（如"亚空间 The Warp"），附录有名词对照表
4. 40K 和中古战锤完全独立，可以跳过一整半只看感兴趣的
